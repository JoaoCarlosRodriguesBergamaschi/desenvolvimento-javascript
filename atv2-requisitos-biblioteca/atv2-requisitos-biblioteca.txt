Estudo de Caso: Sistema de Gerenciamento de Biblioteca

Cenário:

A biblioteca de uma escola deseja modernizar seu sistema de gerenciamento, atualmente baseado em papel, para um
sistema informatizado. Os principais objetivos incluem melhorar a eficiência no controle de empréstimos, facilitar a busca por livros e proporcionar uma experiência mais amigável para os usuários.

Stakeholders:
•Bibliotecários
•Alunos
•Professores

Levantando os requisitos:

Individual:
identificar os requisitos funcionais e não funcionais necessários para o sistema de acordo com o cenário apresentado, pensem no sistema como um todo, em vários aspectos.
